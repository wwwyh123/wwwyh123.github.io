CommonJs

主要用于node.js环境，使用 require 导入模块
两种导出数据的方式:

  • module.exports = value
  • exports.name = value
    其中每个模块内部的this,exports,modules.exports在初始时都指向同一个空对象，该空对象就是当前模块导出的数据
    无论如何修改导出对象，最终导出的都是module.exports的值

    exports是对module.exports的初始引用，仅为了方便给导出对象添加属性，所以不能用exports = value导出数据，但是可以用module.exports = xxx导出数据

import {name as schoolName,slogan,getTel} from './school'

命名导入(对应导出方式：分别导出，统一导出)

命名导入和默认导入可以混合使用
