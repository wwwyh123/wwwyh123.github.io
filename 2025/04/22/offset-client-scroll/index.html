offset,client,scroll

Created2025-04-22|Updated2025-04-24
|Word Count:390|Reading Time:1mins|Post Views:

offset

offsetParent

定义：与当前元素最近的经过定位的父级元素
注意：

  1. 元素自身有fixed定位,offsetParent是null
  2. 元素自身无fixed定位,offsetParent是body
  3. 元素自身无fixed定位,父级元素存在定位，offsetParent是最近的定位的父级元素
  4. body元素的offsetParent是null

offsetWidth,offsetHeight(只读)

offsetWidth = width + border + padding

box.style.width和box.style.height只能获取行内的属性

offsetTop,offsetLeft

offsetTop:当前元素的最上到offsetParent元素的最上的距离
offsetLeft:当前元素的最左到offsetParent元素的最左的距离
e.offsetX 和 e.offsetY 是鼠标事件对象的属性，用于直接获取鼠标点击位置相对于触发事件的元素（如 <canvas>）左上角的坐标。

client

clientWidth = width + padding
clientHeight = height + padding
clientLeft左边框的大小
clientTop上边框的大小
e.clientX 和 e.clientY 是鼠标事件对象 (e) 的属性，表示鼠标点击位置相对于视口的水平和垂直坐标。
注意：

  1. 所有client属性都是只读的
  2. 如果给元素设置display:none;客户端client的属性都为0；
  3. 尽量避免重复访问这些属性

scrollHeight,scrolWidth

scrollHeight:表示元素的总高度，包含由于溢出而无法在网页上的不可见部分
scrollWidth:表示元素的总宽度，包含由于溢出而无法在网页上的不可见部分

  1. 无滚动条的时候，scrollHeight和clientHeight是相等的

scrollTop(可读写)

元素被卷起的高度
当滚动条滚动到内容底部时，符合以下公式
scrollHeight=clientHeight+scrollTop

var docScrollTop = document.documentElement.scrollTop || document.body.scrollTop
