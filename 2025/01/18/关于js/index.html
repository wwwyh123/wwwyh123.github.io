undefined 与 null

console.log(typeof null)//object
console.log(typeof undefined)//undefined

undefined的四种情况

  1. 定义了一个对象但没有赋值
  2. 对象的某个属性不存在
  3. 函数调用少参数
  4. 函数的默认返回值

null

  1. 手动释放内存
  2. 作为函数的参数（此参数不是对象）
  3. 原型链的顶端

forEach

  1. 没有返回值
  2. 不能用break打断
  3. 遍历的是value

map

  1. 有返回值（数组），默认return 是undefined
  2. 接收的参数是一个函数
  3. 不能用break打断

js递归求和

去重

const

  1. const 声明之后必须赋值，否则就报错
  2. 定义的值不能修改，否则报错
  3. 支持let其他属性

var

  1. 声明提升
  2. 变量可覆盖
  3. 没有块级作用域

判断是否是数组的几种方法

  1. Array.isArray()(ES5)
  2. Object.prototype.toString.call()
  3. instanceof Array检查对象是否是 Array 构造函数的实例。
  4. 检查 constructor 属性,但constructor可能被修改 不推荐typeof 运算符
    typeof [] 返回 'object'，无法区分数组和普通对象。

交换a,b的值不用临时变量

  1. 算术运算
  • 加减法
  1. 位运算(适用于整数，浮点数或字符串会失败)
    自反性
    a ^ a = 0 相同数值异或结果为0
    a ^ 0 = a 任何数与0异或结果为其本身
    可逆性
    a = c ^ b
    b = c ^ a
  2. 解构赋值
    [a, b] = [b, a]
