手写call

call方法的作用是：

  1. 修改函数内部的this指向。
  2. 立即调用该函数并传递参数。
    为了保证所有函数都能调用我们的方法，所以把它添加到Function.prototype上
    思路：
  3. 确保thisArg指向一个有效的对象：如果thisArg为null或者undefined，我们将其默认值设为globalThis
  4. 临时将函数作为对象方法：为了改变this,我们将目标函数赋值为thisArg对象的一个临时方法，使用一个唯一的Symbol作为属性名，避免与已有的属性冲突
  5. 调用函数并传递参数
  6. 清理临时属性：删除临时添加到thisArg对象上的方法
  7. 返回函数执行结果
  • 点语法适用于属性名是合法标识符且固定的情况，语法简洁直观。
  • 动态属性名通过方括号语法允许属性名是变量、计算值或包含特殊字符的字符串，灵活性更高

race

该方法接受一个promise可迭代对象作为输入，并返回一个Promise。这个返回的promise会随着第一个promise的敲定而敲定。
如果第一个敲定的 promise 被兑现，那么返回的 promise 也会被兑现；如果第一个敲定的 promise 被拒绝，那么返回的 promise 也会被拒绝。
如果传入的 iterable 为空，返回的 promise 就会一直保持待定状态。如果传入的 iterable 非空但其中没有任何一个 promise 是待定状态，返回的 promise 仍会异步敲定（而不是同步敲定）。
返回一个promise=>判断传入是否为数组=>不是数组则返回错误=>是数组则等待第一个敲定

all

接受一个Promise可迭代对象作为输入，并返回一个Promise。
结果组的顺序和传入组相同。
判断是否为数组 => 空数组直接兑现 => 处理全部兑现 => 处理第一个拒绝
当所有输入的 Promise 都被兑现时，返回的 Promise 也将被兑现（即使传入的是一个空的可迭代对象），并返回一个包含所有兑现值的数组。如果输入的任何 Promise 被拒绝，则返回的 Promise 将被拒绝，并带有第一个被拒绝的原因。

allSettled

返回一个promise;当传入promise都已敲定，即可获取兑现的结果
结果数组[{status:'fulfilled',value:1},{status:'rejected',reason:3}]
